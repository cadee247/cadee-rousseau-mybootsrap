SHOPPING CART

The JavaScript code manages a shopping cart, including adding and removing items, displaying the cart, and calculating the total order amount. Key components are: Variables Initialization: Arrays for item names, prices, quantities, totals, and total order amount. Functions: add_selection(x): Increases item quantity, updates total price, and refreshes the display. remove_selection(x): Decreases item quantity (if greater than 0), updates total price, and refreshes the display. display_all(): Displays the cart items, quantities, prices, and total prices in an HTML table with add/remove buttons. checkout(): Calculates and displays the total order amount. Event Listener: Automatically displays the cart when the webpage loads.

GUESSING GAME

JavaScript Guessing Game: The game generates a random secret number between 1 and 20. The user is prompted to guess the number. Feedback is provided to indicate if the guess is correct, too low, or too high. The game continues until the user guesses the correct number. Key Elements: Secret Number Generation: Uses Math.random() and Math.floor() to create a random number between 1 and 20. while Loop: Repeatedly prompts the user until the correct guess is made. Uses prompt() to get user input and parseInt() to convert it to an integer. The loop provides feedback using if, else if, and else statements.
